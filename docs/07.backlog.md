# Product Backlog - KidCareVax

## Objetivo

Este backlog organiza as funcionalidades necessárias para o MVP do KidCareVax, priorizando as funcionalidades essenciais para o acompanhamento da jornada vacinal infantil.

---

# Epic 01 - Autenticação

Objetivo: permitir que responsáveis acessem a plataforma de forma segura.

## Feature 01 - Cadastro de Responsável

### US-01

Como responsável

Quero criar uma conta

Para acessar a plataforma e gerenciar minhas crianças.

Prioridade: Alta

---

## Feature 02 - Login

### US-02

Como responsável

Quero realizar login utilizando e-mail e senha

Para acessar meus dados cadastrados.

Prioridade: Alta

---

# Epic 02 - Gestão de Crianças

Objetivo: permitir o gerenciamento das crianças vinculadas ao responsável.

## Feature 01 - Cadastro de Criança

### US-03

Como responsável

Quero cadastrar uma criança

Para acompanhar seu histórico vacinal.

Prioridade: Alta

---

## Feature 02 - Listagem de Crianças

### US-04

Como responsável

Quero visualizar todas as crianças cadastradas

Para selecionar aquela que desejo consultar.

Prioridade: Alta

---

## Feature 03 - Detalhes da Criança

### US-05

Como responsável

Quero visualizar os dados da criança

Para acompanhar sua situação vacinal.

Prioridade: Alta

---

# Epic 03 - Histórico Vacinal

Objetivo: permitir o acompanhamento individual das vacinas.

## Feature 01 - Consulta de Histórico

### US-06

Como responsável

Quero visualizar o histórico vacinal da criança

Para acompanhar vacinas aplicadas e pendentes.

Prioridade: Alta

---

## Feature 02 - Status Vacinal

### US-07

Como responsável

Quero identificar rapidamente vacinas aplicadas, pendentes e atrasadas

Para acompanhar a situação vacinal da criança.

Prioridade: Alta

---

## Feature 01 - Listagem de Campanhas

### US-08

Como responsável

Prioridade: Média

---

## Feature 02 - Detalhes da Campanha

### US-09

Prioridade: Média

Objetivo: fornecer visão geral da situação vacinal.

## Feature 01 - Resumo Vacinal

### US-10

Como responsável

Quero visualizar indicadores da situação vacinal das crianças

Para identificar rapidamente pendências.

Prioridade: Média

---

# Epic 06 - Experiência do Usuário

Objetivo: garantir acessibilidade e usabilidade.

## Feature 01 - Responsividade

### US-11

Como responsável

## Feature 02 - Interface Intuitiva

### US-12

# MVP Planejado

As funcionalidades abaixo compõem a primeira versão entregável do produto:

* Cadastro de responsável
* Login
* Cadastro de criança
* Listagem de crianças
* Detalhes da criança
* Histórico vacinal
* Status vacinal
* Visualização de campanhas
* Responsividade

---

# Funcionalidades Futuras

Itens não contemplados no MVP:

* Notificações de vacinas próximas
* Compartilhamento de histórico vacinal
* Integração com calendário vacinal oficial
* Upload de comprovantes de vacinação
* Lembretes automáticos
* Múltiplos responsáveis por criança
